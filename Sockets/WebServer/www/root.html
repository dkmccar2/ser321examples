You can make the following GET requests

  • /file/sample.html -- returns the content of the file sample.html
  • /json -- returns a json of the /random request
  • /random -- returns index.html
  • /multiply -- returns the product of two numbers

    • Example: /multiply?num1=2&num2=2

  • /github -- returns a list of GitHub repos of the supplied user

    • Example: /github?query=users/amehlhase316/repos

Additional Requests

  • /quadraticsolver -- returns the real roots of a quadratic equation: ax^2+bx+c.

    • Example: /quadraticsolver?a=1&b=4&c=3

  • /power -- returns a base raised to an exponent. The result includes 4 decimal places of precision after the decimal point.

    • Example: /power?base=2&exponent=7
